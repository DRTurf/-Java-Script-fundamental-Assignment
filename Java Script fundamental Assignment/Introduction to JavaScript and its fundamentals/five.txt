Comments help the reader understand the code better.
It improves the readability of the code.

Single line comment: Declared using  two back slashes "//"
    Ex- //this is a single line comment

Multi line comment: Declared using a star and an back slash at the 
                    beginning and a back slash star at the end "*/.../*"
    Ex- */this
        is an exmample
        of a multi line comment /*
